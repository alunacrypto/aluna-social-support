Introduction

Welcome to Aluna Social!

We are happy you made to our open source book, throughout this book we plan to help you learn more about the platform, you can contribute to the documentation by submitting changes to our git repository

Aluna Social is a Social Network and Trading Terminal which allows you to manage all your crypto exchange accounts from one place, the platform also provides a telegram bot to interact with your Aluna Account.
